# Nimbus User Manual

This document is to help WordPress administrators update content for the Nimbus website nimbusdance.org. To log in to the WordPress dashboard, go to: nimbusdance.org/wp-admin. If you need to work with a developer, the code for the custom WordPress theme is located here: github.com.

For general Wordpress usage instructions, you an refer to their basic usage documentation
